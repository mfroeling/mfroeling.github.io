SimRotate

SimRotate[din, H ,angle]
rotates the spin system din given the hamiltonian H over angele with phase 90 degrees.

SimRotate[din, H ,angle, phase]
rotates the spin system din given the hamiltonian H over angele with phase. din and H are generated by SimHamiltonian. The angle and phase are defined in degree. The output is a new spinsystem dout.
